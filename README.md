# Random Quote Generator

**Автор:** Крымуков Темирлан

## Описание

Приложение для генерации случайных цитат с возможностью просмотра истории и фильтрации по автору и теме.

### Функциональность:
* Генерация случайных цитат из предопределённого списка.
* Отображение цитаты с указанием автора и темы.
* Сохранение истории сгенерированных цитат.
* Фильтрация истории:
  * по автору;
  * по теме.
* Сохранение истории в JSON‑файл.
* Валидация ввода при добавлении новых цитат.
