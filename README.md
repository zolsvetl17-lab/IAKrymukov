# GitHub User Finder

**Автор:** Крымуков Темирлан

## Описание

Приложение для поиска пользователей GitHub с возможностью сохранения избранных профилей.

### Функциональность:
* Поиск пользователей GitHub по логину.
* Отображение результатов в виде таблицы.
* Добавление пользователей в избранное.
* Сохранение избранных пользователей в JSON‑файл.
* Валидация ввода (проверка на пустое поле).

## Использование API GitHub

Приложение использует GitHub API для поиска пользователей:
* **Endpoint:** `https://api.github.com/search/users`
* **Параметр запроса:** `q` — поисковый запрос.
* **Ограничения:** 10 запросов в минуту для неавторизованных запросов.
